## 🔍 关键改动点

### 1. 时间特征提取（UTC时区）
```python
def _extract_time_features(self, timestamps: np.ndarray) -> np.ndarray:
    dt = datetime.fromtimestamp(ts, tz=timezone.utc)  # UTC时区
    time_feats[i, 0] = dt.year - 2000
    time_feats[i, 1] = dt.month
    # ... 其他特征
    return np.clip(time_feats, min_val, max_val)  # 严格限制范围
```

### 2. 维度扩展
```python
# 原来
self._buf_user_int = np.zeros((B, self.user_int_schema.total_dim), dtype=np.int64)

# 现在
self._buf_user_int = np.zeros((B, self.user_int_schema.total_dim + 15), dtype=np.int64)
```

### 3. Schema 自动更新

```python
time_feature_vocab_sizes = [101, 13, 32, 24, 60, 7, 2, 101, 13, 32, 24, 60, 7, 2, 5]
for i, vs in enumerate(time_feature_vocab_sizes):
    self.user_int_schema.add(10000 + i, 1)
    self.user_int_vocab_sizes.append(vs)
```
